# 1. 什么是递归

简单来说就是，当函数自己调用自己本身时，就会发生递归现象。  

举例如下：

```python
def sum_number(n):
    '''
    使用递归来实现 1+2+3……+n
    :param n:
    :return: 求和的结果
    '''
    if n==1:
        return 1
    temp = sum_number(n-1)
    return n+temp

def main():
    num = int(input("请输入你要计算的数字："))
    print(sum_number(num))

if __name__=="__main__":
    main()
```
